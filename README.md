# 📚 Smart Book Recommendation System

## 🚀 Live Demo
👉 https://smartbook-recommendation-system.streamlit.app/
## 📌 Project Overview

The **Smart Book Recommendation System** is an intelligent web application that helps users discover books based on preferences, popularity, and similarity.  
It uses Machine Learning techniques to recommend books through a simple and interactive interface.

This project is built using **Python, Pandas, NumPy, Scikit-learn, and Streamlit**.

---

## ✨ Features

- 📖 Book recommendations based on user selection  
- 🔍 Search books by title  
- ⭐ Popular books suggestions  
- 🤖 Similarity-based recommendation engine  
- 💻 Interactive Streamlit web interface  
- ⚡ Fast and user-friendly design  

---

## 🛠️ Technologies Used

- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Streamlit
